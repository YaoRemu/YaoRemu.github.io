Vue不提倡操作 DOM

jQuery帮我们封装了一些 DOM操作，它是用来操作 DOM 的； 而Vue是帮我们免除操作 DOM 的这个过程，所以 Vue 中一般不使用 jQuery

在 Vue 中，使用事件绑定机制，为元素指定处理函数的时候，如果加了小括号，就可以给函数传参了

some()： 根据指定的条件进行判断，如果返回true，some() 就会被终止，此案例中推荐使用 some()

findIndex()： 专门用来查找索引
