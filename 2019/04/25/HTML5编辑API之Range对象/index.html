HTML5编辑API之Range对象

 · 2019-4-25 · 次阅读

课程概要

  • Range对象
    • Range对象的基本概念
    • selectNode / SelectNodeContents / deleteContents 方法
    • setStart / setEND setStartBefore / setStartAfter /setEndBefore / setENDAfter 方法

Range对象的基本概念

一个 Range对象 代表页面上的一段连续区域。 通过 Range对象 ， 可以获取 或 修改网页上的任何区域

控制节点的方法

SelectNode方法

Range对象 的 SelectNode方法 用于将 Range对象 的起点指定为某个节点的起点，将 Range对象 的终点指定为该节点的终点，使 Range对象 所代表的的区域中 包含该节点，该方法并不将该节点设定为高亮选取状态。

SelectNodeContents方法

Range对象 的 SelectNodeContents方法 用于将 Range对象 的起点，指定为某个节点中的所有内容的起点，将 Range对象 的终点指定为该节点中所有内容的终点，使 Range对象 代表的区域中，包含该节点中的所有内容

deleteContents方法

Range对象 的 deleteContents方法 用于将 Range对象 包含的内容从页面中进行删除
