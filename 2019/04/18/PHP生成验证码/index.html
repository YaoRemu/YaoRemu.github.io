相关函数解释

  • session_start()：启动session绘画（session变量用于存储有关用户绘画的信息并且可供应用程序中的所有页面使用）

注：使用session时，要保证在使用他之前，没有向浏览器输出过任何内容，所以通常被放置在第一行

  • header(‘content-type:image/png’)：向客户端发送原始的HTTP报头

PHP图形处理函数：

  • imagecreatetruecolor(int x, int y)：建立一副大小为x、y的黑色图像（默认为黑色），如果想改变颜色则需要用填充颜色函数imagefill($img,0,0,$color)
  • imagecolorallocate($image, 255, 255, 255)：为指定图像分配颜色
  • imagefill($image, 0, 0, $bgcolor)：填充颜色函数
  • imagestring($image,$fontsize,$x,$y,$fontcontent,$fontcolor)：水平的画一行字符串
  • imagesetpixel($image, rand(1,99), rand(1,29), $pointcolor)：或一个单一的像素（eg：点）

  • imageline($image, rand(1,99), rand(1,29),rand(1,99), rand(1,29) ,$linecolor)：画一条线段

  • imagepng($image)：以PNG格式将图像输出到浏览器或文件

  • imagedestroy($image)：销毁图像
